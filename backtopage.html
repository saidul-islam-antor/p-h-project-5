Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a parent element listens for events on its child elements, even if those child elements are dynamically added later.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM (Document Object Model) is a mechanism where an event that occurs on a nested element first triggers on that element and then propagates upward to its parent elements.
